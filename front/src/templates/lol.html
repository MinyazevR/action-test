    Свитч и хаб (Основы компьютерных сетей)

    Сеть демонстрирует различие в работе хаба и свитча.

    Маршрутизатор

    Маршрутизатор объединяет IP сети.

    NAT 2 (основы компьютерных сетей)

    NAT (Network Address Translation (преобразование сетевых адресов) - механизм, позволяющий на лету изменять IP адрес отправителя.

    Кольцо из 3-х маршрутизаторов (Основы компьютерных сетей)

    Поле TTL в IP заголовке защищает сеть от постоянного циркулирования IP пакетов.

    Первый и последний адрес IP сети (Основы компьютерных сетей)

    Демонстрация сети, где у хостов назначен первый и последний адрес сети.

    TCP - установка соединения 1 (Основы компьютерных сетей)

    Успешная установка TCP соединения (3-х разовое рукопожатие).

    ICMP - сеть недоступна (Основы компьютерных сетей)

    ICMP сообщение "Сеть недоступна" генерируется, когда хоста или маршрутизатор не знает маршрута до IP сети, в которой находится IP адрес назначения.

    ICMP - хост недоступен (Основы компьютерных сетей)

    ICMP сообщение "Хост недоступен" генерируется в случае, когда маршрутизатор или хост не может доставить пакет до хоста с IP адресом назначения.

    Групповой трафик- UDP (Основы компьютерных сетей )

    Сеть демонстрирует работу группового трафика (Multicast).

    STP (Основы компьютерных сетей)

    STP (Spanning Tree Protocol) — канальный протокол. Основной задачей STP является устранение петель в топологии произвольной сети Ethernet.
